# Portfolio Website

React + TypeScript + Vite を Docker 環境で開発するポートフォリオサイトです。  
Node.js をローカルにインストールせず、Docker コンテナ上で開発できます。

---

## Tech Stack

- React
- TypeScript
- Vite
- Docker / Docker Compose
- Node.js 20

---

## Prerequisites

- Docker Desktop がインストールされていること
